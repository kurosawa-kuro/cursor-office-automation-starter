## なぜ Cursor なのか

- **ターミナル不要**（黒い画面を触らない）= 非エンジニア向き
- **無料の Hobby 枠**で始められる（クレカ不要）
- ファイルを見ながら**日本語でお願い**できる

<!--
ナレーション：
「AI ツールは他にもありますが、Claude Code などは黒い画面＝ターミナル操作が前提で、初めての方には敷居が高い。
Cursor は画面のクリックと日本語入力だけで完結し、しかも無料枠で始められます。だから入門に最適なんです。」
所要：18秒
-->

## 【デモ③】教材を ZIP でダウンロードして開く

- 展開した `..._main` フォルダを `Open Folder` で開く

<!--
