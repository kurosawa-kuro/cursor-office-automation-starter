## Cursor って何？（身近な道具と比べる）

| 道具 | 何をする | 入力 |
|---|---|---|
| Excel | 表で計算 | セルに数式 |
| Word/メモ帳 | 文章を書く | キーボード |
| VS Code | 高機能メモ帳 | キーボード |
| **★Cursor** | **VS Code + AI** | **日本語で頼む** |

> むずかしく考えず「**AI に日本語で頼めるメモ帳**」

<!--
ナレーション：
「Excel が表計算の道具なら、Cursor は『AI に文章やデータ処理を頼む道具』。VS Code というメモ帳に AI を組み込んだものです。」
所要：20秒
-->

## なぜ Cursor なのか

- **ターミナル不要**（黒い画面を触らない）= 非エンジニア向き
- **無料の Hobby 枠**で始められる（クレカ不要）
- ファイルを見ながら**日本語でお願い**できる

<!--
ナレーション：
「Claude Code などは黒い画面＝ターミナルが前提で敷居が高い。Cursor は画面操作と日本語入力だけ、しかも無料枠で始められます。」
所要：18秒
-->
